# fr-cyber-incident-analysis-2026
2026年に発生したフランスANTS（市民権局）のデータ流出事案を題材にした、インタラクティブなインフォグラフィックです。Chart.jsによるデータの可視化と、Gemini APIを活用したパーソナライズド・リスク分析機能を実装。　Interactive analysis of the 2026 ANTS data breach. Features Chart.js visuals and a Gemini AI assistant for secure, practical cybersecurity education.
